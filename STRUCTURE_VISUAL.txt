SCALABILITY:
============

Add new section:
1. Create src/components/sections/NewSection.jsx
2. Add to src/components/sections/index.js
3. Import in App.jsx
4. Done! ✅

Add new UI component:
1. Create src/components/ui/NewComponent.jsx
2. Add to src/components/ui/index.js
3. Import anywhere
4. Done! ✅

Expand calculator:
1. Add files to src/components/calculator/
2. Keep isolated from other features
3. Done! ✅
